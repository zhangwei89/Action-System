# 附录 A 行动系统工具箱

下面是 12 个可以直接使用的模板和工具。每个模板都按"易上手、可执行、有反馈"的标准设计，可以直接复制使用。

---

## 1. SMART 目标模板

**S（Specific 具体）**：目标要具体，不是"我要变好"，是"我要 X"。

**M（Measurable 可衡量）**：能衡量，不是"我要读完一些书"，是"30 本"。

**A（Attainable 可达成）**：可达成，不是"我要成为首富"，是"我要减重 8 公斤"。

**R（Relevant 相关）**：相关，目标要和长期愿景相关。

**T（Time-bound 有时限）**：有时限，不是"我要学英语"，是"6 个月内达到听懂访谈的水平"。

**使用示例**：

- 目标：6 个月内减重 8 公斤。
- 具体（Specific）：通过饮食控制 + 每周 3 次跑步。
- 衡量（Measurable）：每周记录体重，月底对比。
- 可达成（Attainable）：每月减 1.3 公斤，节奏健康。
- 相关（Relevant）：健康是生活质量的基础。
- 有时限（Time-bound）：6 个月。

---

## 2. OKR 目标模板

**O（Objective 目标）**：定性，描述方向。有挑战性、有启发性。

**KR（Key Results 关键结果）**：定量，3-5 个可衡量的结果。

**使用示例**：

- Objective：成为公司公认的技术专家。
- KR1：本季度主导 2 个核心项目的技术方案。
- KR2：本季度在团队内做 3 次技术分享。
- KR3：本季度在技术社区发表 2 篇深度文章。
- KR4：本季度通过 1 项高级技术认证。

OKR 不和绩效考核挂钩，重在挑战和成长。完成 0.7-0.8 是合理状态，完成 1.0 说明不够有挑战。

## 模板使用建议

1. **不要一次全用。** 挑 1-2 个最需要的先用，跑顺了再加。
2. **改成你的版本。** 模板只是起点，根据你的实际改。
3. **用工具承载。** Notion、飞书、Obsidian、Excel、纸笔都行，关键是跑起来。
4. **定期评估。** 每季度看哪些模板在用、哪些没用，把没用的砍掉。
5. **持续迭代。** 模板是死的，你的实践是活的。把有用的部分留下，没用的换掉。

## 工具搭配建议

不同工作流可以搭配不同的工具：

**轻量级（适合个人新手）**：

- 待办：滴答清单 / Todoist
- 笔记：Notion / 语雀
- 复盘：Notion 数据库
- 时间：Toggl / RescueTime

**中量级（适合深度使用）**：

- 待办：Things 3 / TickTick
- 笔记：Obsidian / Logseq
- 项目：Notion + Airtable
- 时间：Toggl + 日历
- 习惯：Streaks / Habitica

**重量级（适合团队/复杂项目）**：

- 项目：Asana / Jira / 飞书项目
- 文档：Notion / 飞书文档
- 沟通：Slack / 飞书 / 钉钉
- 数据：Airtable / Looker
- 自动化：Zapier / n8n

工具是中性的，先选一个跑起来，跑顺了再换。
